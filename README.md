SmartFunds SACCO Management System

SmartFunds SACCO is a web-based financial management system designed for SACCO (Savings and Credit Cooperative Organizations) members. The system enables users to manage savings accounts, handle loan applications and disbursements, track shares, and perform withdrawal operations. It provides a complete digital platform for cooperative financial operations with user authentication, account management, and transaction tracking capabilities.

Main Function Points
User authentication and account login/logout functionality
Savings account management (deposit, view, and withdraw savings)
Share management (purchase, view, and withdraw shares)
Loan management system supporting both short-term and long-term loans
Loan eligibility checking and approval workflow
Account information viewing and updating
Dashboard for monitoring account status and transactions
Multiple transaction types (deposits, withdrawals, loan collection)
Technology Stack
Backend: PHP (72%)
Frontend: HTML (28%)
Database: MySQL (smartfund_sacco.sql)
Architecture: Server-side rendered web application
